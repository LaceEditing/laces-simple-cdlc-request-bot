üéÆ User Commands

!tokens Check your VIP token balance
!list Get link to the queue page
!queue Alias for !list
!song See current/next song
!position See your position in the queue
!myqueue See your queue position
!remove Remove your last request
!help Show available commands

üîß Mod Commands

!givevip <user> <amount> Give VIP tokens to a user
!clear Clear entire queue (broadcaster only)

üß™ Test Commands (Mod Only)

!testbits <amount> Simulate a bits donation (default: 250)
!testsuperchat <amount> Simulate a Super Chat in $ (default: $5)

These commands let you test the VIP token system without real purchases.
